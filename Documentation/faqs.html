PlayTubeVideo Features

Where can I access the admin panel?

Login to the site using your admin credentials, then visit https://www.YOURSITE.com/video-admin

Where can I edit the stylesheet?

You can edit stylesheet go to path ./static/css/styles.css

Where can I edit the theme coloring and fonts?

Login to the site using your admin credentials, then visit https://www.YOURSITE.com/video-admin/designs/color

How to edit & customize email templates which are sent for users?

Login to the site using your admin credentials, then you can find the email template in https://www.YOURSITE.com/video-admin/mail/templates

How to edit the PRO packages page?

Login to the site using your admin credentials, then you can edit the packages from https://www.YOURSITE.com/video-admin/payments/packages

How to manage Member Roles?

Login to the site using your admin credentials, then you can manage member roles from https://www.YOURSITE.com/video-admin/levels.

How can I manage the Site menu?

Login to the site using your admin credentials, then you manage the menu from https://www.YOURSITE.com/video-admin/menus

Where is the page template files located?

Login to the site using your admin credentials, then you can create or edit the pages from https://www.YOURSITE.com/video-admin/pages

How to manage Advertisementss?

You can do that from Admin Panel > Advertisement > Website Advertisement.

How can I manage Browse page content settings?

You can do that from https://www.YOURSITE.com/video-admin/pages/content.

How can I manage Comments?

You can do that from https://www.YOURSITE.com/video-admin/comments

How can I manage blogs/blog categories and member role settings?

Login to the site using your admin credentials, then you can do that from Admin Panel > Blogs settings.

How can I manage the Playlist ?

Login to the site using your admin credentials, then you can do that from https://www.YOURSITE.com/video-admin/playlists.

How can I manage Channels?

Login to the site using your admin credentials, then you can do that Admin Panel > Channels settings.

Where I can view Sold videos, Import videos and setup member role settings for Videos?

Login to the site using your admin credentials, then you can do that Admin Panel > Videos settings.

Where is the User settings page?

Login to the site using your admin credentials, then you con do that Admin Panel > Users settings.

Where can I view my earnings?

You can view that from https://www.YOURSITE.com/video-admin/earnings.

How can I set up website language?

Create a new language from https://www.YOURSITE.com/video-admin/languages page, enable and submit the changes. Once you have clicked the Make Default button, the selected language will display as the default language for the site.

How can I create new slides?

Create new slide from https://www.YOURSITE.com/video-admin/slideshow. For buttons and links on the slide, make the enable button Yes.

When I am trying to recharge my balance for advertisement, I get ”Client Authentication failed”. How to solve it?

The PayPal configuration are wrong. Please make sure the access keys are correct and make sure it's live or sandbox in Admin Panel > payments > Payments Settings.
