Paypal Api Key

1. Login to Your PayPal Developer Account

  • Login to https://developer.paypal.com using your live PayPal account credentials.

2. Click on My Apps & Credentials and then click on Live

3. Click on Create App

4. Write App name and click on Create App button

5. Copy Client ID and Client Secret

6. Click on Add Webhook button

7. In Webhook Url field enter URL:

  • https://YOURSITEURL/payment/IPN
  • then check All events option
  • then click Save button.
